Legend and definitions

Size code (A–F)
ICAO aerodrome reference code by wingspan. A stand's max code must be equal to or larger than the aircraft's. A320 is code C, B777 is E, A380 is F.
Contact vs remote
Contact stand has a jet bridge at the terminal. A remote stand sits on the apron and needs bus boarding plus extra handling.
Schengen / non-Schengen
Passport-control routing. A Schengen arrival needs a Schengen-side contact stand, or a zone change is forced.
Buffer
Minutes between this aircraft clearing the stand and the next aircraft needing it. Target is 25 minutes or more.
Baggage belt
Each stand feeds a baggage belt. Moving the aircraft can re-route bags to a different belt.
Ground handler
The company servicing the turnaround. A handler already at capacity is an operational risk.
Confidence
High, Medium, or Low, derived from the computed score and the constraints, not assigned by hand.
Redirect / cancel
The airport does not cancel an airline's flight. The tool flags candidates and recommends escalation to the airline and ATC.

How it works: hard constraints filter out illegal stands; weighted soft preferences rank the rest. Assigning a stand removes it from the pool, so other flights re-rank. The duty manager decides.
